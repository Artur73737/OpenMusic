### Quick Start

<details>
<summary><b>1. Technical Backend</b></summary>

```bash
cd backend
pip install -r requirements.txt
uvicorn music_mcp.api_server:app --port 8001
```
</details>

<details>
<summary><b>2. Studio Frontend</b></summary>

```bash
cd frontend
npm install
npm run dev
```
</details>
